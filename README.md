# GlobalVoice-Design

Aqui será o ambiente dos artefatos de design e prototipagem do GlobalVoice.

Repositório com duas frentes complementares (embora estejam no mesmo diretório, a organização separa os fluxos):

- **Fluxo de dados**: diagramas de arquitetura e fluxo de informações do sistema.
- **Design de telas**: wireframes, protótipos de alta fidelidade e especificações visuais.

## Visão geral

A ideia é separar os contextos:

- **Fluxo de dados** para documentar a arquitetura do sistema, a comunicação entre os componentes (frontend, backend, APIs) e o percurso das informações (áudio, texto, tradução).
- **Design de telas** para centralizar a identidade visual (paleta de cores, tipografia, componentes) e as interfaces propostas (tela principal, chat, barra de ferramentas, etc.).

Assim cada frente evolui com seus próprios arquivos (imagens, PDFs, arquivos editáveis do Figma ou Draw.io), sem misturar os artefatos conceituais com os visuais.

Para os designs de tela: abrir as imagens PNG/PDF diretamente no navegador ou em qualquer visualizador de imagens.
